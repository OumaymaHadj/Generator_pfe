## Reporting Issues

We use GitHub Issues as the official bug tracker for the Vue App. Here are some advices for our users that want to report an issue:

1. Make sure that you are using the latest version of the Vue App. Check the CHANGELOG from your dashboard on our [website](https://www.creative-tim.com/product/vue-argon-dashboard?ref=readme-vad).
2. Providing us reproducible steps for the issue will shorten the time it takes for it to be fixed.
3. Some issues may be browser specific, so specifying in what browser you encountered the issue might help.
